# Projeto Iguaçu Solar

**Iguaçu Solar** é um projeto desenvolvido como parte do curso de **Engenharia de Software (UNINTER)**.  
O objetivo é promover o conhecimento sobre **energia solar** e seus benefícios para a população de **Nova Iguaçu (RJ)**.

## Tecnologias utilizadas
- **HTML5**
- **CSS3**
- **JavaScript**
